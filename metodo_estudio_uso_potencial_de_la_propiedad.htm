El estudio del uso potencial de una propiedad para uso agrícola

La palabra "agricultura" se refiere aquí a todas las técnicas que se emplean para proporcionarle a las plantas las condiciones ideales para que manifiesten su potencial y cumplan con la función o utilidad que se ha previsto, ya sea producir alimentos, materias primas para el procesamiento de otros productos, protección de superficies, seguridad de áreas, superficies para práctica de deportes y recreación, embellecimiento y ornato, entre otras.

Las técnicas agrícolas deben ser adaptadas a las condiciones de suelo y clima. Las técnicas agrícolas se pueden agrupar en dos grupos:

  • Las técnicas de preparo de suelo
  • Las técnicas para el cultivo de plantas.

También es importante destacar que para este estudio el término "propiedad" se emplea para identificar el espacio que contiene la tierra en la que se va a sembrar. Los tipos de propiedad que podemos encontrar son diversos, entre ellos: las No Urbanas, tales comos fincas, haciendas, conucos, fundos, hatos; y las Urbanas, tales como casas, edificios, centros comerciales y profesionales, edificios gubernamentales, colegios, universidades, plazas, parques, islas de avenidas y aceras, campos deportivos, aeropuertos, etc.


El Estudio del uso potencial de la propiedad para uso agrícola, se realiza durante el Diagnóstico en la Fase de Identificación.  Este es el primer paso en la dirección de una agricultura ecológica correcta.


Este estudio consiste en el levantamiento e interpretación de toda la información que nos permite conocer las características y condiciones de la propiedad, las parcelas de tierra que la comprenden y la región en donde se encuentra. Toda la información levantada e interpretada durante el estudio, es guardada en forma de informes, gráficos, tablas, fotos de campo, fotografías aéreas, planos, etc.


La información levantada es interpretada para determinar el uso y manejo mas racional que puede dársele a cada una de las parcelas que integran la propiedad. Esta información organizada nos permite establecer las relaciones entre los recursos que intervendrán directamente en la producción. Cada parcela es clasificada por sus características y atributos específicos, y a partir de allí, se generan recomendaciones de acuerdo al uso más adecuado para cada una de ellas de una manera práctica en lo que se refiere a: 

  • Método de conservación del suelo
  • Tipos de correctivos del suelo
  • Tipo de riego
  • Tipo de drenaje
  • Recomendaciones de Cultivos
  • Recomendaciones de fertilización

 Beneficios socioeconómicos
